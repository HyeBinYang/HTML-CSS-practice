# Tag : Box vs Item

| Tag  | 특정                                                                                                                              | 종류                                                                   |
| ---- | --------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------- |
| Box  | - 나중에 CSS로 스타일링은 가능하지만 HTML에서는 보이지 않음<br />- 여러가지 item들을 그룹화해서 재사용 가능한 item들이 모아져있음 | header, nav, footer, main, section, article, aside, div, span, form 등 |
| Item | 실제 사용자의 눈에 보임                                                                                                           | a, input, label, img, button, h1, p 등                                 |
